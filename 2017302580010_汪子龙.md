NQ/N(Q/R+d~poll~)=R/(1+d~poll~R/Q)



**Problem 19**

| 时间        | 事件                            |
| ----------- | ------------------------------- |
| 0           | A、B开始传输                    |
| 245         | A、B检测碰撞                    |
| 293         | A、B结束拥塞信号传输            |
| 293+245=538 | B最后一比特到达A；A检测空闲线路 |
| 538+96=634  | A开始传输                       |
| 293+512=805 | B返回第二步                     |
|             | A传输到达B                      |

由于A的重传在B的预定重传时间805+96=901之前到达B，所以B在A重传时不进行传输，因此A和B不发生碰撞，因此指数退避算法中出现的因子512足够大。
